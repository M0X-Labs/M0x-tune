# Core  fine-tuning dependencies
#
# NOTE: install torch separately FIRST from the CUDA-specific index that matches your
# GPU driver, e.g.:
#   pip install torch --index-url https://download.pytorch.org/whl/cu124
# (see scripts/install_deps.py / detect_hardware.py for automatic CUDA-tag detection).
#
# We intentionally use the `huggingface` extra here, not `windows`/`cuXXX-torchYYY`:
# `[windows]` pins a loose, wheel-unconstrained xformers requirement that, when no
# prebuilt wheel matches your (python, torch, cuda) combo, silently falls back to compiling
# from source -- which requires a full MSVC + Windows SDK toolchain and fails without one.
# `[huggingface]` pulls transformers/trl/peft/accelerate/datasets ('s own
# tested-compatible version ranges) without requiring xformers;  falls back to
# PyTorch's built-in SDPA attention automatically when xformers isn't installed.
#
# IMPORTANT: pinned to an exact release (NOT `git+...@main`). Installing straight off
# 's main branch means every fresh install (a new device, a rebuilt Colab/Kaggle
# session, a re-run of setup.sh weeks later) can silently resolve to a *different*
#  commit, which in turn drags in different trl/transformers/peft/accelerate
# versions each time (see /pyproject.toml's `huggingface`/`huggingfacenotorch`
# extras -- those ranges are only loosely bounded). That's the root cause of "works on
# one machine, crashes with a totally different error on another": two installs a week
# apart were never actually running the same library versions. Pinning an exact version
# here makes every install reproducible.
#
# The transitive versions below are copied verbatim from  2026.6.9's own
# `huggingfacenotorch` extra (/pyproject.toml) so they're documented/auditable
# here rather than buried in a third-party package's metadata, and so uv/pip can't
# silently widen them if a future requirement in this file nudges the resolver. When
# bumping the pinned  version, re-copy these ranges from the new release's
# pyproject.toml `huggingfacenotorch` extra rather than hand-editing them.
